# MacBookLighter
MacBook light controller (especially with sound) based on openFrameworks.

## Requirements
- openFrameworks 0.9.3+
- ofxDisplayBrightness
- ofxKeyboardBacklightBrightness
- ofxUI
- ofxXmlSettings
